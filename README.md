Save The System merupakan sebuah **game desktop berbasis Java (NetBeans)** yang dirancang sebagai **simulasi pengambilan keputusan dalam pengelolaan sistem/server**.  
Dalam game ini, pemain berperan sebagai seorang **admin sistem** yang bertanggung jawab menjaga kondisi sistem agar tetap berjalan dengan stabil dan aman.

Game ini bertujuan untuk melatih pemain dalam memahami pentingnya **keseimbangan antara stability, performance, dan security**, di mana setiap keputusan yang diambil akan memberikan dampak langsung terhadap kondisi sistem.

## ğŸ“Œ Deskripsi Program
Pada permainan ini, pemain akan dihadapkan pada berbagai permasalahan sistem, seperti **server overload**, gangguan performa, hingga ancaman keamanan.  
Setiap ronde akan menampilkan sebuah *problem* yang berbeda, dan pemain harus memilih tindakan yang paling tepat untuk menanganinya.

Setiap aksi yang dipilih memiliki konsekuensi tertentu terhadap kondisi sistem, sehingga pemain dituntut untuk **berpikir strategis, mempertimbangkan risiko, dan memahami trade-off dari setiap keputusan**.

### Tujuan utama permainan:
- Menjaga keseimbangan **Stability**, **Performance**, dan **Security**
- Mengumpulkan **score** setinggi mungkin melalui keputusan yang tepat
- Menyimpan dan menampilkan hasil permainan pada **Leaderboard**
